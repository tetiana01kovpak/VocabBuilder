# VocabBuilder

Web app for building and training an English–Ukrainian vocabulary: personal
dictionary with progress tracking, recommended words from other users, and an
interactive training mode.

## Tech stack

Next.js 14 (App Router) · TypeScript · Redux Toolkit + redux-persist · Formik +
Yup · @tanstack/react-table · axios · react-hot-toast · CSS Modules

## Getting started

```bash
npm install
cp .env.example .env.local
npm run dev
```

Open http://localhost:3000.
